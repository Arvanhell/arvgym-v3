# ARV GYM v.3.0 ⚡

**ARV GYM** is a high-performance training log and biometric tracker designed for athletes who demand precision. Built with a focus on speed, local data sovereignty, and mobile-first accessibility.

## 🚀 Core Features
* **Dynamic User Management:** Custom multi-profile system without cloud dependency (Local Storage).
* **Biometric Engine:** Real-time BMI calculation and body weight tracking.
* **Strength Archive:** Persistent workout logging with RPE (Rate of Perceived Exertion) metrics.
* **Neural Aesthetics:** High-contrast "Cyber-Gym" interface with Orbitron typography.
* **Bilingual Support:** Full English (EN) and Polish (PL) localization.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox/Grid)
* **Logic:** Vanilla JavaScript (ES6+)
* **Persistence:** LocalStorage API
* **Deployment:** Netlify Continuous Deployment

## 📊 Biometry & Focus
The application calculates intensity based on training goals:
- **Hypertrophy:** 90s recovery focus.
- **Power:** 180s recovery focus.
- **Shred:** 60s recovery focus.
