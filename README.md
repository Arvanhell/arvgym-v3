## 🚀 The Feature Set

### 👤 Hardcore User Management
Multi-profile system with zero latency. Every athlete gets their own sandbox for logs and bio-metrics, baked straight into `LocalStorage`.
* **New:** Refreshed Profile & ID icons (👤🆔) for that clean, high-tier look.

### 🧬 Bio-Metric Engine
Real-time BMI crunching and body-type tracking. Dynamic color-coding keeps you posted on your current form—from Lean to Muscular.

### 🏛️ Strength Archive & Manual Inject
Full-stack workout history powered by **RPE** (Rate of Perceived Exertion) metrics. 
* **The Fix:** Migrated the **Manual Injector** to the History Modal. Keeps the main dashboard clean while letting you hot-fix your logs on the fly.

### ⏲️ Tactical Rest Timer
Hard-coded recovery intervals based on your training objective:
* **Hypertrophy:** 90s (The Sweet Spot)
* **Power:** 180s (Max Effort)
* **Shred:** 60s (Metabolic Burn)
